About This Model

This CNN was trained on the Kaggle Brain Tumor MRI dataset ( GitHub Repository ) containing 7,023 images, split into four classes:

  • Glioma: Arises from glial cells—often malignant, found in cerebral hemispheres.
  • Meningioma: Originates from the meninges—usually benign, occurs near brain surface.
  • No Tumor: Normal brain scans with no detectable tumor tissue.
  • Pituitary: Develops in the pituitary gland—can affect hormone regulation.

• Architecture: 5 convolutional layers + dense head
• Framework: TensorFlow 2.10 & Keras API
• Test‐set accuracy: ~92%

• Model weights stored in models/brain_tumor_cnn_classifier.keras
• JSON structure in models/CNN_structure.json
• See README for setup, training logs, and usage.
